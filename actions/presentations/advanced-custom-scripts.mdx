title: "Advanced: Custom Scripts"
---

Add custom scripts to your presentation slides for personalization.

**Add script to presentation**

1. Go to Engagement tab
2. Scroll to "Double's Script"
3. Add custom instructions (optional)
   - Example: "Mention their company name if they tell you"
   - Example: "Use technical terminology for engineers"
4. Save

**Script behavior**

- Applies to opening message only
- Influences tone and content
- Doesn't interrupt presentation flow
- Can be updated anytime

## **Quick Reference**

| **Task** | **Steps** |
| --- | --- |
| **Create presentation** | Upload deck → Link Double → AI Instructions → Engagement messages |
| **Embed on website** | Copy embed code → Paste in HTML/React → Set height |
| **Share standalone** | Copy presentation link → Share with visitors |
| **Edit existing** | Click presentation → Edit Setup or Engagement → Save |
| **Test presentation** | Ask your Double to offer it → Click through all slides |
| **Pause/delete** | Click presentation → Pause or Delete → Confirm |
