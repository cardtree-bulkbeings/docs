---
title: "Managing Presentations"
---

### **View all presentations**

Go to Actions → Presentations to see:

- Presentation name and thumbnail
- Linked Double
- Creation date
- Status (Published, Paused, Draft)

**Edit a presentation**

1. Click the presentation
2. Edit Presentation Setup or Engagement tabs
3. Changes apply immediately
4. Live presentations update in real time

**Pause a presentation**

1. Click the presentation
2. Click Pause
3. Presentation stops being offered by your Double
4. Existing links still work (visitors can access)

**Delete a presentation**

1. Click the presentation
2. Click Delete
3. Confirm deletion
4. Cannot be undone — presentation is permanently removed
