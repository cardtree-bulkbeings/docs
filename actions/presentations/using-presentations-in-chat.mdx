---
title: "Using Presentations in Chat"
---

**How visitors trigger presentations**

- Conversation-based: Visitor asks about features → Double offers presentation
- Action-based: Visitor clicks "See demo" button → Presentation starts
- Direct link: Share presentation link directly with visitors

**During presentation**

- Visitors control slide navigation (Next, Previous, End)
- Conversation history is paused (can resume after)
- Visitor can ask questions — returns to chat mode temporarily
- Voice conversations work (visitor can speak with your Double)
