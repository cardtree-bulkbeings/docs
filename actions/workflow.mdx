title: "Workflow"
---

Coordinate multi-step tasks and connect different apps into one smooth automation flow. Build complex logic with branching, conditions, and routing.

**What Workflows do**

- Chain multiple apps together (Double → Gmail → Sheets → Router → multiple branches)
- Add conditional logic (if/then branches, routing rules)
- Build multi-step sequences with delays, wait conditions, and custom code
- Automate complex processes without manual intervention

**Tools available in Workflows**

- **Apps** — Calendly, Gmail, Google Sheets, Google Calendar, Salesforce, Zoho CRM, and more
- **Logic & Control** — Condition (branch based on conditions), Router (route through named branches), Delay (pause for a set time), Wait For (pause until user responds)
- **Custom** — Code (run custom JavaScript), HTTP Request (call any API)

**Best for**

- Multi-step processes (capture lead → check CRM → send email → log to sheet)
- Complex routing and conditional automation
- Integrating more than one app in a single flow
- Custom business logic and API calls

**How Workflows work**

1. Go to Actions → Workflows
2. Click Create Workflow
3. Start with a trigger (On Chat Start, or other conditions)
4. Add steps by connecting apps and tools
5. Set up branches and conditions as needed
6. Test your workflow and publish when ready

**What to know**

- Workflows run automatically based on triggers you set
- Each workflow can branch into multiple paths
- Multiple tools and apps can be chained in one flow

## **Quick Comparison**

| **Feature** | **Presentations** | **Skills** | **Workflows** |
| :-- | :-- | :-- | :-- |
| **Setup** | Upload slides, add instructions | Configure one app or native capability | Chain multiple apps with logic |
| **Complexity** | Single task (present) | Single task (action) | Multi-step sequences |
| **Triggers** | Visitor intent (what they ask) | Conversation trigger (keywords, intent) | Multiple triggers & conditions |
| **Typical use** | Demo, pitch, training | Send email, show image, fill form | Multi-app automation, routing |
| **Setup time** | ~5 minutes | ~5 minutes | ~15-30 minutes |
