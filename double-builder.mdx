title: "Double Builder "
---

The Double Builder is your visual editor for customizing how your Agent page looks and feels. Every change reflects instantly in the live preview.

To open the builder, go to your Double from Home and click **Edit**, or open it directly from your Double's agent preview page.

The builder has two panels — a **left sidebar** for navigation and a **right panel** that opens when you click any element to edit it. Click the **✏️ edit icon** on any section in the preview to open its settings directly.

### **Identity**

Controls how your profile appears on the agent page — both the side panel (right side) and the center section (left side).

Click the **✏️ edit icon** on the identity panel to open Identity settings. There are two tabs: **Side Identity** and **Center Identity**.

### **Side Identity**

The side panel shows your profile on the right side of the agent page.

- **Identity Panel** — set your display Name, Title, Company, and Logo
- **Avatar** — toggle your profile photo on or off
- **Logo** — toggle your company logo on or off
- **Text** — set the font and text color for the side panel
- **Background** — set the side panel background as Solid or Gradient, with a custom color

### **Center Identity**

The center section sits above the chat input on the left side of the agent page.

- **Hero title** — the main heading visitors see (e.g. your name)
- **Hero description** — your subtitle (e.g. title and company)
- **Avatar** — toggle your profile photo on or off
- **Banner BG** — upload a custom banner background image
- **Voice Orb** — choose between **Dot** or **Pulse** style, and customize:
  - Bounce speed
  - Orbit speed
  - Dot count
  - Color
- **Text** — set font, title color, and description color
- **Background** — set the banner background color (Solid or Gradient)

### **Greetings & Quick Actions**

Controls the text and conversation starters visitors see on the agent page.

Click the **✏️ edit icon** on the greeting or Quick Actions section to open **Interaction** settings. There are two tabs: **Quick Actions** and **Greetings**.

### **Quick Actions**

Starter prompts that appear as clickable buttons on the agent page — guiding visitors to start a conversation.

- Add a new prompt by typing in the **Add your suggestion** field and clicking **Add**
- Each prompt can be set to **Chat** or **Voice** mode
- Set the default mode for new prompts using **New pill mode**
- Click **Apply to all** to set all existing prompts to the same mode
- Reorder prompts by dragging, or delete them with **×**

### **Greetings**

The headline and subtext shown above the chat input.

- **Double Greetings** — edit the main greeting title and description (e.g. "Chat with my AI Double." / "It's trained on my data and voice.")
- **Call to action** — add up to 3 phrases that cycle through the "Ask me anything" placeholder with a typing effect
- **Text** — set font, greeting title color, greeting description color, and text alignment

### **Style**

Controls the overall visual theme of the agent page.

Click the **✏️ paint brush icon** at the bottom of the preview to open **Style** settings.

- **Skins** — choose **Light** or **Dark** mode, and **Monotone** or **Tinted** widgets. Pick an accent color from the palette.
- **Backgrounds** — set a custom background for the agent page
- **Voice Orb** — customize the voice orb style, speed, and color (also accessible from Center Identity)
- **Text** — set global font and text styling

### **What to know**

- All changes preview live — no need to save separately
- The builder is split into **Double** and **Card** sections in the left sidebar — customizations here apply to the Agent page only
- To customize the Card page, switch to **Card** in the sidebar
