Native Skills are built-in capabilities that work immediately — no app connection needed. Set them up once, and your Double triggers them during conversations based on keywords, intent, or user requests.

#### **There are six native skills:**

- Display Image — Show images when triggered in chat
- Share PDF — Display PDF documents when triggered
- Dynamic Response — Reply with specific messages when triggered
- Questionnaire — Create interactive forms and surveys
- Embed social content — Surface Instagram, YouTube, and LinkedIn posts
- Link Embed — Display website links as rich preview cards

## **Display Image**

Trigger your Double to display an image when a visitor uses a specific keyword or phrase.

#### **What it does**

- Uploads a single image
- Triggers when visitor mentions keywords
- Shows the image inline in chat
- Works with JPG, PNG, and other common image formats

#### **Best for**

- Portfolio pieces and work samples
- Product screenshots or diagrams
- Comparison visuals
- Certificates or credentials

#### **How to set up Display Image**

1. Go to Actions → Skills
2. Under Native skills, click Configure on Display Image
3. Click \+ Add new
4. Select your Double from the dropdown
5. Upload an image (click or drag and drop)
6. Add keywords in the Description field — these trigger the image display (e.g., "portfolio", "work samples")
7. In Agent Prompt, describe when the image should appear (e.g., "When the visitor asks to see your portfolio")
8. Click Create Skill

#### **What to know**

- Only one image per Display Image skill
- Create multiple skills for multiple images
- Keywords are case-insensitive
- The Double shows the image whenever the trigger phrase is mentioned

## **Share PDF**

Trigger your Double to display a PDF document when a visitor uses specific keywords or phrases.

**What it does**

- Uploads a single PDF document
- Triggers when visitor mentions keywords
- Displays the PDF inline for preview and download
- Works with all PDF types (guides, whitepapers, case studies)

**Best for**

- Case studies and white papers
- Product guides and documentation
- Pricing documents
- Legal agreements or contracts
- Research reports

**How to set up Share PDF**

1. Go to Actions → Skills
2. Under Native skills, click Configure on Share PDF
3. Click \+ Add new
4. Select your Double from the dropdown
5. Upload a PDF file
6. Add trigger phrases in the Description field (e.g., "case study", "pricing guide", "white paper")
7. In Agent Prompt, describe when to surface this PDF (e.g., "When the visitor asks for pricing information")
8. Click Create Skill

**What to know**

- One PDF per skill
- Create multiple skills for multiple documents
- Visitors can preview and download the PDF
- Trigger phrases work with partial matches

## **Dynamic Response**

Set your Double to reply with a specific message whenever a trigger condition is met.

**What it does**

- Responds with exact text when keywords are mentioned
- Customizable message for each trigger
- Supports both text and formatted responses
- Works across chat and voice

**Best for**

- Frequently asked questions with standard answers
- Policy statements or disclaimers
- Contact information responses
- Promotional messages

**How to set up Dynamic Response**

1. Go to Actions → Skills
2. Under Native skills, click Configure on Dynamic Response
3. Click \+ Add new
4. Select your Double from the dropdown
5. Add trigger phrases (e.g., "hours", "contact", "pricing")
6. Enter the response message in the text field
7. In Agent Prompt, add context for when to use this response (optional)
8. Click Create Skill

**What to know**

- Multiple trigger phrases can share one response
- Each skill can have one or more trigger phrases
- The Double responds exactly as written
- Best for straightforward Q&A, not nuanced conversations

## **Questionnaire**

Create interactive forms and surveys that your Double administers during conversations.

**What it does**

- Builds guided interview questions or surveys
- Collects structured responses from visitors
- Supports multiple question types (text, multiple choice, ratings)
- Integrates with Contacts for lead capture

**Best for**

- Lead qualification forms
- Customer feedback surveys
- Product preference questionnaires
- Interview-style information gathering

**How to set up Questionnaire**

1. Go to Actions → Skills
2. Under Native skills, click Configure on Questionnaire
3. Click \+ Add new
4. Select your Double
5. Name your questionnaire (e.g., "Product Interest Survey")
6. Add questions one by one:
   - Enter the question text
   - Choose question type (text input, multiple choice, rating scale)
   - For multiple choice, add all answer options
7. Set whether responses are optional or required
8. Click Create Skill

**What to know**

- Questions appear as a guided form in chat
- Responses are saved and linked to the contact
- You can reorder questions anytime
- Each question type shows different input methods to visitors

## **Embed social content**

Surface Instagram, YouTube, and LinkedIn posts directly in chat when visitors ask about them.

**What it does**

- Embeds saved Instagram posts, YouTube videos, or LinkedIn articles
- Triggers when visitor mentions keywords
- Shows rich media inline (videos play in chat, posts display with images)
- Works with public content only

**Best for**

- Showcasing recent posts or videos
- Highlighting customer testimonials (LinkedIn)
- Sharing behind-the-scenes content
- Promoting case studies or success stories
- Video tutorials or explainers

**How to set up Embed social content**

1. Go to Actions → Skills
2. Under Native skills, click Configure on Embed social content
3. Click \+ Add new
4. Select your Double from the dropdown
5. Paste the link to your public Instagram post, YouTube video, or LinkedIn article
6. Add trigger phrases (e.g., "latest post", "check my videos", "recent content")
7. In Agent Prompt, describe when to surface this content (e.g., "When the visitor asks to see recent updates")
8. Click Create Skill

**What to know**

- Only public posts/videos embed successfully
- Instagram Stories and private videos won't work
- Video embeds play directly in chat
- Create separate skills for different posts or videos
- Trigger phrases help visitors discover content naturally

**Examples**

- Instagram post: "When they ask 'show me your work' or 'what have you posted'"
- YouTube video: "When they ask 'do you have tutorials' or 'show me how'"
- LinkedIn article: "When they ask 'case studies' or 'success stories'"

## **Link Embed**

Display website links as rich preview cards with automatic title and description fetching.

**What it does**

- Pastes a URL and auto-fetches title and description
- Shows link as a rich preview card (not just plain text)
- Triggers when visitor mentions keywords
- Supports optional custom title and description

**Best for**

- Promoting blog posts or articles
- Sharing resource links (tools, templates)
- Directing to landing pages
- Highlighting guides or documentation
- Driving traffic to specific pages

**How to set up Link Embed**

1. Go to Actions → Skills
2. Under Native skills, click Configure on Link Embed
3. Click \+ Add new
4. Select your Double from the dropdown
5. Paste the URL in Website URL field
6. Review the auto-fetched Card Title and Description
7. Optionally customize the title or description (auto-fetch can be overridden)
8. Add trigger phrases (e.g., "blog", "resources", "article")
9. (Optional) Add Custom Instructions — notes for your Double on how to reference this link
10. Click Create Skill

**What to know**

- Title and description auto-populate from the webpage
- You can edit them to be more specific
- The link card shows a preview, not just a bare URL
- Visitors can click the card to visit the page
- Works best with pages that have good meta descriptions

**Examples**

- Blog post: Trigger phrase "read my latest blog" → shows article with preview
- Template: Trigger phrase "download template" → shows resource link with description
- Landing page: Trigger phrase "learn more" → shows dedicated page link

## **Getting Started**

**Start with one:** Pick the skill that matches your most common visitor question.

**Use keywords wisely:** Trigger phrases should match how visitors naturally ask for your content.

**Stack multiple skills:** You can run all six native skills on the same Double — each handles different content types.

**Test and refine:** Publish your skills and watch conversations to see which trigger phrases work best.
