## Quickly connect an app or teach your Double a single task so it executes immediately during a conversation.

**Two types of Skills are available**

- **Native Skills** — Built-in capabilities with no app connection needed (Display Image, Share PDF, Dynamic Response, Questionnaire, Embed social Content, Link Embed)
- **App Skills** — Connect one external app (Gmail, Calendar, Sheets, Salesforce, Zoho CRM) to add new abilities

**What Skills do**

- Execute a single action instantly when triggered
- Run without needing to set up multi-step workflows
- Work with both chat and voice conversations

**Best for**

- Quick actions (send email, display a document, show an image)
- Gathering information (forms, questionnaires)
- Single app integrations (email a form response, add to a spreadsheet)

**How Skills are organized**

1. Go to Actions → Skills
2. Browse My Skills — Your active skills across all Doubles
3. Use categories, starred items, and active filters to manage them

**What to know**

- Each skill performs one task
- Multiple skills can be set up on the same Double
- Both native and app skills appear in the same list
