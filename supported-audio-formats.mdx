## **Supported Audio Formats**

Use the right audio format when uploading a voice sample to get the best clone quality.

**Recommended format**

- WAV - Most stable format. Recommended for best cloning results.

**Supported formats**

- WAV
- MP3 (may cause issues in some cases - convert to WAV if cloning fails)
- Other common audio formats are supported but WAV is preferred

**File limits**

- Maximum duration: 30 seconds
- Maximum file size: 5 MB
