---
title: "Toubleshooting"
---

**Presentation not starting?**

- Check that the Presentation is published
- Verify AI Instruction is clear and specific
- Test the trigger phrase with your Double
- Ensure presentation is in Published status

**Voice not playing?**

- Check browser speakers/volume
- Verify the linked Double has a voice configured
- Test microphone permissions (Chrome → Settings → Privacy)
- Try a different browser

**Slides not displaying?**

- Verify deck uploaded successfully (check file size)
- Test with a different browser
- Clear browser cache and refresh
- Try re-uploading the deck

**Embedded presentation not loading?**

- Check iframe code is correct (all parameters present)
- Verify presentation ID is correct
- Ensure iframe has allow="microphone; camera"
- Test in incognito/private window (check for ad blockers)

**Visitor can't advance slides?**

- Clarify in opening message: "Click Next to move to the next slide"
- Check that presentation is Published (not Paused)
- Test navigation buttons yourself
- Ensure browser isn't blocking clicks
